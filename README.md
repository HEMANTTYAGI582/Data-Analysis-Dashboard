# Alok-Collection-Data-Analysis (Interactive Dashboard using MS Excel)
## Project Objective
THe Alok Collcetion wants to create an annual sales report for 2022. So that, the owner of the Alok Collection can understand their customers and grow more sales is 2023 & 2024.

## Dataset used 

<a href="https://github.com/HEMANTTYAGI582/Data-Analysis-Dashboard/raw/refs/heads/main/Alock%20Collection%20DashBoard%20Report.xlsx">Alok Collection Dataset</a>

## Questions (KPIs)
- Sales vs. Orders: Compare the sales and orders using a single chart.
- Peak Performance: Which month got the highest sales and orders?
- Customer Demographics: Who purchased more—men or women in 2022?
- Order Tracking: What are the different order statuses in 2022?
- Geographic Distribution: List the top 10 states contributing to the sales.
- User Profiles: What is the relation between age and gender based on the number of orders?
- Channel Analysis: Which channel is contributing to maximum sales?
- Product Performance: What is the highest selling category?
